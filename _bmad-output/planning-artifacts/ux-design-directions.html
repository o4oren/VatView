Design Notes — ATC Staffing Pattern

  • Letter badges: D=Delivery, G=Ground, T=Tower, A=Approach, I=ATIS (inspired by VATSIM Radar)
  • Only staffed positions show badges — unstaffed airports show ICAO only (clean, no visual noise)
  • Traffic counts: green up-arrow = departures, red down-arrow = arrivals (with number)
  • Badges appear at appropriate zoom levels — hidden at global zoom, revealed as user zooms in
  • Airport dot glows blue when any ATC position is staffed; dim grey when no ATC online

Recommendation

  • Direction 2 (Balanced HUD) as the default Phase 1 experience
  • Direction 6 (Clean Modern) validates light theme — same components, different tokens
  • Direction 5 (Dark Instrument) reserved for Phase 2 aviation theme collection
  • ATC letter badges (D/G/T/A/I) — universal pattern across all directions. Only show for staffed positions.
  • Traffic counts (dep/arr arrows) — appear at appropriate zoom levels alongside airport badges
